# School Attendance Management System

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**A modern, fully responsive web app for efficient school attendance tracking.** Built with pure HTML/CSS/JS – no backend needed. Perfect for teachers to manage classes, mark daily attendance, and generate insightful reports.

## 🌟 Key Features

- **Student CRUD**: Seamlessly add/edit/delete students (ID, name, class). Persistent storage via LocalStorage.
- **Daily Marking**: Pick date → mark Present/Absent per student → instant save & update.
- **Smart Reports**: Custom date-range analysis with attendance %, sorted tables, & Chart.js bar graphs visualizing performance.
- **Mobile-First Responsive**: Adapts perfectly to any screen (desktop/tablet/phone) using CSS Clamp, Grid/Flexbox, horizontal-scroll tables.
- **Intuitive UX**: Gradient themes, modals, success/error toasts, hover animations. Offline-ready.

**Tech**: HTML5 | CSS3 (responsive utils) | Vanilla JS ES6+ | Chart.js CDN | LocalStorage/JSON.

**Live Demo**: Open `index.html` or `npx live-server attendance-system`. Includes 15 sample students (9A-12B).

**Zero Setup**: Instant run, no installs. Extendable for exports/backend.

Future: CSV export, auth, bulk import.
